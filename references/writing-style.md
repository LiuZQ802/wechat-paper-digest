# 写作要求：具体，不让读者猜

核心标准只有一条：**读者扫一眼一句话，要能立刻知道这句话在说什么具体的事**，不需要脑补、不需要猜你到底想说啥。模糊的概括句是默认会被拒收的，除非紧跟着具体内容把它落实了。

## 检查方法

写完 `article.md` 后，逐句过一遍，对每一句问：**如果删掉这句话，读者会损失什么具体信息？** 如果答案是"没有损失，这句话只是营造了一种'厉害/重要/有效'的语气"，这句话就要么删掉，要么补上让它具体的内容。

第二个检查点：句子里提到的数字/结论，**是不是来自论文里同一个实验**？把"在A实验里得到的结论"和"在B实验里得到的数字"揉进同一句话、共用一个笼统的主语（比如"三个语料上的实验显示标注效率提升27%"，但27%其实只在其中一个语料上测过），是常见的精简过头导致的失真，不只是表达模糊，是事实错误。压缩内容时要先确认每个数字背后对应的是哪个具体实验，不能为了让一句话覆盖更多结论就把不同实验的产出混在一起。

## 典型的"模糊空话"——尽量别这么写

这些词本身不是禁用词，但**单独出现、后面不跟具体内容**时几乎总是在说"我也不知道具体是什么，但听起来很厉害"：

- "显著提升/显著改善" → 提升到多少？从多少到多少？
- "有效解决/有效降低" → 降低了多少？怎么做到的？
- "在一定程度上/某种程度上" → 到底是什么程度，能不能换成数字或者范围？
- "表现良好/效果不错" → 用什么指标衡量的，数值是多少？
- "具有重要意义/具有重要价值" → 意义具体是什么？说出那个意义本身，而不是说"有意义"这件事。
- "新颖的方法/创新性的设计" → 新在哪？跟旧方法具体差在哪一步？
- "多个场景/多种情况下" → 哪几个场景？能不能直接点名？
- **"三个语料/两种方法/几项实验"这类"数字+类别名词"，光有数字不算具体**——数字只交代了数量，没交代是哪几个。只要句子里能塞下，就把每一项的名字都列出来（"三个语料"→"Weibo、CLUENER、人民日报三个语料"）；如果上一句或同一段已经点过名，后面可以简称，但**第一次出现时必须点名**，不能让数字本身充当具体性的证明。
- **"全部数据/所有样本/整个数据集"这类"全称表述"，同样要落实到具体数量**——论文里几乎总能查到这个"全部"对应的确切数字（通常就在附近的表格里），查到了就直接写上去（"全部训练数据"→"全部1730条训练样本"），不要让"全部/所有"这种词替代真实的数字。
- "很朴素/很实在/很扎实"这类对实验设计本身的主观评价词 → 直接描述这个实验设计是什么（比如对比了哪两种方式、跑了几组、每组多少样本），不需要先评价一句"朴素""实在"

## 对照例子

| 模糊（不要这样） | 具体（这样写） |
|---|---|
| 该方法效率更高 | 推理只需29.5分钟、花费0.154美元，是GPT-4o（9.12美元）的约1/60 |
| 模型表现有显著提升 | F1从0.3483提升到0.4487，提升了0.1004 |
| 在多个语料上都表现良好 | 在Weibo、CLUENER、人民日报三个语料上都优于随机采样基线 |
| 作者提出了一种新颖的采样策略 | 过去的采样方法只看模型当前这一轮的预测，这篇论文新增了"看历史多轮预测有多不稳定"这一维度 |
| 这一发现具有重要意义 | （直接说发现本身是什么，不需要再加一句"这很重要"） |
| 该框架综合考虑了多种因素 | 框架同时用了三个指标：温度A、压力B、湿度C（具体点名是哪几个） |

## 适用范围

- **内容导读**、各小节正文、图说，全部按这个标准检查。
- 数字、专有名词、论文里的具体例子/案例，能用就用，不要为了"读起来顺"而磨掉细节。
- 这不等于句子要写得啰嗦——具体和简洁不矛盾，"F1从0.35到0.45"比"模型效果有所改善但仍有提升空间"短得多，但信息量大得多。
- 小标题允许用贴切的类比或成语，但整篇文章的语气应贴近科研通讯/学术简讯，不追求"生动有趣"本身；类比之后必须落回具体内容，不能让类比代替信息，也不能为了显得活泼而牺牲准确。

## 用词：用规则判断，不要查黑名单

网络流行语一直在冒新词，列多少个都列不完，不能靠"遇到一个禁一个"。判断要靠下面这条规则，不是靠记一份名单。

**对每一个"感觉有点潮/有点梗"的词，连续问两个问题：**

1. **来源测试**：这个词是规范书面语、词典词汇、成语里就有的，还是从网络流行语、缩写黑话、谐音梗、表情包、弹幕用语来的？后者不用，不管它现在多流行。
2. **替换测试**：把这个词换成一个更朴素的同义说法，意思会不会变？如果不会变——说明这个词存在的理由只是"显得有梗/显得会聊"，不是传达额外信息——换成朴素说法。如果换了之后确实丢失了准确性（朴素说法说不清那个意思），说明这个词是必要的，可以留。

两条测试都通过（书面语/成语 + 替换会丢信息）才保留，否则一律换成直白描述。

**这条规则不是为了排除生动表达**——成语、贴切的比喻、自创但意思一望即知的小标题都欢迎。可操作的判断尺度："这个词会不会被一家严肃科普媒体（果壳、中国国家地理、澎湃科技这类）的编辑划掉？"会被划掉的就别用。

（曾经被划掉的例子：反复横跳、苦活、难啃、很朴素也很实在、真香、绝绝子、拿捏——这些不是因为上了黑名单才不能用，是因为按上面两条测试都没通过。以后遇到新词，自己跑一遍这两条测试就行，不需要等着把新词加进这份例子列表。）

## 连贯性：章节之间要有桥梁句，不能是孤立的知识点堆砌

每一节衔接处，读者应该能看出"为什么读完上一节、紧接着要读这一节"，而不是感觉在读几个互不相关的知识点列表。

- 每一节的**开头或上一节的结尾**，写一句承上启下的话，把上一节留下的问题/结论和这一节要讲的内容连起来（例如："要解决这个问题，论文提出了……"、"光挑对样本还不够，标注本身也要……"）。
- 自查方法：把所有小标题单独抽出来看一遍，应该能读出一条完整的逻辑线（提出问题 → 怎么解决 → 怎么验证 → 验证结果说明什么），而不是一堆并列的、可以任意打乱顺序的标题。
- 内容导读里提前点出文章的逻辑主线（要解决几个问题、对应几个方案），后面每节展开时呼应这条主线，读者读到一半时始终知道"现在讲的是主线里的哪一步"。

## 标点与表达：避免一眼看出来是AI写的

- **少用分号"；"**：中文流行写作很少在一句话里用分号并列分句，分号堆出来的句子读着像翻译腔/AI腔。能拆成两句话就拆成两句话，或者改用逗号、句号断开。
- **少用破折号"——"做插入语/强调**：偶尔用一次没问题，但一篇文章里到处都是"——"，读起来会有明显的机器味。能用句号断句、或把插入内容改写成单独一句话的，优先这么处理。
- **避免套话式过渡句反复出现**：比如"这意味着……""换句话说……""不仅……还……"，偶尔用一次没问题，但同一篇文章里用上好几次，就成了AI写作的标志性口癖，读者会觉得在读模板。
- 自查方法：写完后数一下全文有几个"；"和"——"，超过一两个就挑出来看看是不是真的需要——大部分时候改写成两句话会更自然、更像人写的。

## 内容要素：核心科学问题、方法、实验设计、结果、意义都要讲清楚

不要求每一节都展开讲全部要素，但整篇文章读完，这五件事读者都应该能说清楚：

1. **核心科学问题**：这篇论文到底想解决什么问题？现有方法卡在哪？（不是"提出了一个新方法"这种空泛说法，要说清楚旧方法具体哪里不行）
2. **方法**：论文用了什么具体手段解决这个问题？关键机制是什么？
3. **实验设计**：在什么数据/场景上验证的？跟谁比、比了什么指标？（哪怕只用一两句话带过，也不能完全不提——"在三个语料上做了对比实验"比完全不交代实验设计要好）
4. **结果**：验证出来的具体数字/结论是什么？
5. **意义**：这个结果对这个领域/这类问题意味着什么？（不是泛泛的"意义重大"，是说清楚意义具体是什么，参照前面"模糊空话"那条）

写完之后自查：这五件事里，有没有哪一件完全没提到？如果有，是不是该补一句。

## 公众号发布标题：准确概括论文内容，不靠悬念/拟人博眼球

标题（`文献荐读 | <短标题>`）跟正文遵循同一条"具体性"标准：读者看到标题就该知道这篇论文研究了什么、有什么具体发现，不需要靠悬念或人格化包装才能明白在说什么。这是一份论文推荐栏目，语气应贴近科研通讯/学术简讯，不是营销号标题。

- 标题围绕论文的**真实研究对象、方法或核心发现**展开，可以用一个数字或论文里的具体概念做关键词（"16万篇论文的研究区标注"、"面向地理空间表格数据的TabPFN改进"），信息本身就是吸引力的来源，不需要额外包一层比喻或拟人化的壳。
- 不使用悬念式提问、"AI+动词"拟人句式、网络热梗式的标题结构（如"XX出动""这招""藏得有多深"）。可以用简洁贴切的类比，但前提是它准确对应论文内容，去掉类比后标题依然完整、不依赖悬念才成立。
- 反例（悬念/拟人代替具体信息，不要这样写）：
  - 文献荐读 | AI侦探出动：16万篇论文里，"研究区"都藏在哪儿？
  - 文献荐读 | 选样本也要"记仇"？这篇论文靠这招把标注效率提了27%
- 对照写法（同样具体，但不靠悬念/拟人）：
  - 文献荐读 | 16万篇论文的研究区标注实践：地理学"研究区"该如何界定？
  - 文献荐读 | 一种基于历史预测波动的主动学习采样策略，标注效率提升27%
- 反例（只有空泛形容，没有具体内容，同样不能用）：文献荐读 | 一篇关于AI和地理研究的好论文
- 同一篇论文可以想多个标题方向，但都要落在"准确概括论文实际内容"这条线上，不要为了吸引点击改写成悬念句或拟人句。
